# Веб-карта точек Dodo Pizza по всему миру

**Проект создан исключительно в ознакомительных целях и не связан с брендом Dodo Pizza.**

этот проект отображает **интерактивную карту точек Dodo Pizza** по всему миру с детальной информацией о пиццериях и их доходах, так как в компании додо верят, что прозрачность превыше всего. Также реализован удобный поиск по пиццериям для быстрого нахождения нужного объекта.

## Стек

-   **сборщик:** Vite
-   **фреймворк:** всеми любимый React
-   **состояние и хранение данных:** React Query и Zustand
-   **карта:** красивая тёмная карта благодаря MapLibre GL
-   **UI:** секси интерфейс с Tailwind CSS и Shadcn/UI
-   **API:** использованы Старое публичное API Dodo Pizza и Global API

## Установка

```bash
git clone https://github.com/damirtag/dodoweb.git
cd dodoweb
npm install
npm run dev
```
